Question

A company has a large networked computer system.

Some of its data is non-sensitive data that would cause no risk to the company if accessed. Some data, however, is sensitive, such as the company’s financial records and documents that contain trade secrets and personal information about employees or clients.

Data corruption can result in data loss.

Outline two ways in which access to sensitive data can be managed.

[4]
a.

Outline two ways to improve the security of the company’s network.

[4]
b.

Explain how corrupted data files can be recovered.

[4]
c.

The company is considering implementing a virtual private network (VPN).

Explain one benefit to the company of using a VPN.

[3]
d.

Markscheme

Award [4 max] 

Each user should have appropriate access rights;
Managers, employees and clients/customers are permitted to access different parts of the data;

Sensitive data can be protected by locking it with a password;
preventing unauthorised access who doesn't have this;

Administrator should record who has logged on and from which computer / and for how long;
to discourage security violations/ to avoid undesirable events from occurring;

File systems should be encrypted (as it passes throughout a network / resides on computers);
to make data unusable if accessed by unauthorized user;

Each computer should have access rights depending on its location;
Logged in computers should not be unattended;

In order to prevent intruders/strangers from accessing the company's network;
the router is set to accept only specific MAC addresses;

to ensure the network is protected against new threats such as malware;

Examiners report

Candidates often confused the requirements of this question with those of part 10(b). Candidates who were able to demonstrate an understanding of how access to sensitive data could be managed, which means how access may be restricted to data, using methods such as access rights or encryption, achieved marks. Candidates who simply discussed network security methods, such as usernames and passwords, generally did not.

a.

Candidates were generally able to gain some marks on this question by outlining methods of improving security on the company's network. Answers relating to controlling access to the network itself, such as through the use of whitelists, or usernames and passwords, generally achieved marks.

b.

This question asked candidates to explain how corrupted data files can be recovered. Candidates who simply discussed taking backups did not answer the question. Those who went on to say how these backups could be used to recover the corrupted data, or who recognized the need for this data to be up to date, generally scored marks.

c.

The vast majority of candidates were able to recognize the benefit of a VPN, with many two- and three-mark answers. However, the question asked about the benefit to the company, with candidates generally recognizing this as being increased security or enabling remote access. However, candidates whose answers related too much to the benefits for the individual users lost marks.
